
## 1. Install iODBC

Using MacPorts:
```
sudo port install libiodbc
```

## 1. Get psqlODBC

```
wget https://ftp.postgresql.org/pub/odbc/versions/src/psqlodbc-09.03.0400.tar.gz
tar xzf psqlodbc-09.03.0400.tar.gz 
cd psqlodbc
./configure --with-iodbc --enable-pthreads
make
sudo make install

```
